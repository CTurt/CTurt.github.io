Making an NES game

I tried to make an NES game based on the N64 game Buckbumble.

This was obviously a very ambitious first project for the platform. The system has a lot of difficult limitations to work with, which I increasingly became aware of as the project developed; these limitations are explained in detail here and here.

Regardless, I got a single level, no enemy, working prototype; at which point I decided I'd had enough.
